\# Design Guidelines

\#\# Overall Style  
\- Simple like HuckToons.com  
\- Clean spacing, readable type  
\- Minimal distractions — clarity sells  
\- Focus on the three core features as separate sections

\#\# Color Palette (recommended)  
Primary: \#1A73E8 (Google Blue)    
Secondary: \#0A4297 (Deep Trust Blue)    
Accent: \#4CAF50 (Green — success action)    
Background: \#FFFFFF    
Text Dark: \#111111    
Gray UI: \#F4F6F8  

Note: Light theme default.

\#\# Typography  
Headings: Inter, Bold    
Body: Inter / Open Sans, Regular    
Buttons: Semi-bold, all text readable

Font Sizes:  
\- H1: 40–52px  
\- H2: 26–32px  
\- Paragraph: 16–18px  
\- Nav: 15–16px

\#\# Layout  
\- Generous white space  
\- Sections use full-width rows  
\- Responsive mobile priority  
\- Hero headline large and punchy

\#\# Buttons  
Style:  
\- Rounded 6–8px  
\- Solid primary color (\#1A73E8)  
\- Hover: Slight darken \+ shadow

CTA Examples:  
\- “Start Free Trial”  
\- “Book Demo”  
\- “See How It Works”

\#\# Component Library to Build  
\- Header \+ Nav  
\- Hero block  
\- 3 Feature Cards (main selling points)  
\- Testimonial block (later)  
\- Pricing table  
\- Footer minimal

\#\# Lead Dashboard UI  
Minimal table interface:  
\- Columns: Name | Phone | Inquiry | Date | Status  
\- Status with pill colors  
\- Top-right "Export CSV" button
